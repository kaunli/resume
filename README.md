## 📖 Overview

This project transforms a traditional resume into an interactive web experience. Rather than relying on a standard PDF layout, the website emphasizes clean typography, thoughtful information hierarchy, and responsive design to create a professional portfolio piece.

The resume includes:

- Professional summary
- Work experience
- Education
- Technical skills
- Certifications
- Featured projects
- Printable PDF version
---
## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
